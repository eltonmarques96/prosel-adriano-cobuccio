# Saas Template
Template with basic CRUD template using NodeJS with OvernightJS (backend), ReactJS (frontend) and Expo (mobile)

### Functionalites
- Create User
- Login
- Validate User
- Forgot Password

## Tools
### Backend
#### Tools
- NodeJS
- Typescript
- Overnight JS (Routes)
- Jest (Tests)
- TypeORM
- Postgres
- Pino
- SQLite (For unit Tests)
### Frontend
#### Tools
- ReactJS with Vite
- Typescript
- AdminLTE
- Axios
### Mobile
#### Tools
- React Native
- Expo
- Typescript
- Style Components
- Axios
